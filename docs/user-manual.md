# 魚川トリミ 使い方ガイド

デスクトップにいる魚川トリミ(うおかわ とりみ)の使い方です。
むずかしい操作はありません。**基本は、話しかけるだけ**です。

---

## はじめて起動したとき

最初の1回だけ、**API キーの設定**をお願いします(トリミの頭脳に Claude を使うためのカギです)。

1. 起動すると設定画面が出ます。画面の中の「**Anthropic Console を開く**」からカギを取得します。
2. カギを貼り付けて「**接続テスト**」→「**保存して始める**」。
3. トリミが挨拶したら準備完了です。

> カギはお使いのパソコンの中だけに暗号化して保存され、Anthropic 以外には送られません。

---

## トリミに話しかける

**マウスをトリミに重ねる**と、胸元に入力欄とボタンが現れます(マウスを離すと消えます)。

- **文字で話す**:「**話しかける…**」の欄をクリックして入力し、**Enter**。
- **声で話す**:マイクボタン(下記)を使います。

トリミの返事は、**頭の上の吹き出し**に出ます。吹き出しは少し経つと自動で消えます(クリックでも消せます)。

---

## 下のボタン(左から)

| ボタン | できること |
|--------|-----------|
| 🎤 **マイク** | **クリックで音声入力の ON/OFF**(呼びかけたら聞く)。**押し続けている間だけ録音**することもできます(押して話す)。聞いているあいだは**緑に光ります**。 |
| 🔊 **音量** | マウスを重ねると音量つまみが出ます。**クリックでミュート**(トリミの声を消す)。 |
| ☕ **離席** | 押すとトリミが**後ろを向いて待ちます**(「ちょっとまってね」)。このあいだ声は聞きません。もう一度押すと戻ります。 |
| ⚙ **設定** | 設定を開きます(下記)。 |
| 👋 **じゃあね** | 「じゃあね」して**タスクバーに最小化**します。**完全に終わるわけではなく**、タスクバーのボタンをクリックすると戻ってきます。 |

> マイクが緑に光っているときは「聞いている」合図です。マウスを離していても光ったままなので、いつでも分かります。

---

## 動かす・しまう

- **動かす**:トリミを**ドラッグ**して好きな場所へ。
- **しまう/戻す**:👋「じゃあね」で**タスクバーに最小化**。**タスクバーのボタンをクリック**すると戻ってきます。

---

## 設定でできること

⚙ 設定ボタンから開きます。

- **話しかけてくる頻度**:トリミが自分から話しかける頻度(しない / 控えめ / ふつう)。
- **見た目の調整**:立ち姿の高さ・距離・向きなど。
- **API キーを変更**:カギを入れ直すとき。
- **位置をリセット**:トリミを最初の場所に戻す。
- **このアプリについて / クレジット**:バージョンや音声の権利表記。

---

## 完全に終了する

- **トリミを右クリック → 「アプリを終了」**、または
- **タスクバーのボタンを右クリック → 「ウィンドウを閉じる」**

(「じゃあね」ボタンは“しまう(最小化)”だけなので、完全に終わるときはこちらです。)

---

## 困ったとき

| こんなとき | 確認すること |
|-----------|------------|
| 返事が来ない / 「調子悪いみたい」と言う | API キーの残高、またはインターネット接続。設定 →「API キーを変更」で見直せます。 |
| 声が出ない | 🔊 音量がミュートになっていないか。 |
| 声を聞き取ってくれない | マイクがつながっているか、パソコン側でマイクの使用が許可されているか。 |
| 見当たらなくなった | 「じゃあね」で最小化中かも。**タスクバーのボタンをクリック**すると戻ってきます。 |

---

## データとプライバシー

- 会話の記憶や設定は、**アプリ(exe)と同じ場所の `data/` フォルダ**に、そのまま読める形(平文 JSON)で保存されます。フォルダごとバックアップ・引っ越し・削除ができます。
- API キーだけは暗号化して保存されます。
- 外部に送るのは **Claude への会話テキストだけ**です。利用状況の収集などは一切ありません。
